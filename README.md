# Hyprside

This repository contains packages, scripts and configurations to build the Hyprside Linux-based operating system. If you want to know more what this project is about, read [here](https://github.com/hyprside/.github/blob/main/profile/README.md)

To build Hyprside you need:
- Docker
- [Hyprpacker](https://github.com/hyprside/hyprpacker)

Hyprpacker is the main build tool used to orchestrate the build of the operating system.
